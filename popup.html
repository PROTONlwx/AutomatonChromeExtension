Documentation

Automaton can click/input on your page and wait-for/repeat those tasks. Use this page to tell the Automaton what to do, and the automaton will do them step by step.

Four buttons:

  • Execute: Start running the Automaton.
  • Save: Save the current Automaton (Must select buttons again if page was closed/refreshed).
  • Clear: Remove all actions from Automaton.
  • Add Action: Append a new specified action.

Five actions:

  • Click: An action that click on the page.
    1. Click button Choose Target
    2. Right click target button on page
    3. Click Add to action menu item
  • Input: An action fills input area on the page.
    1. Click button Choose Target
    2. Right click target button on page
    3. Click Add to action menu item
    4. Type content in Input
  • Wait: Wait a while before moving on.
    1. Enter the number of seconds to wait.
  • Goto: Continue to some step. (Max one Goto in Automaton)
    1. Enter which step to continue executing.
  • Break: Stop after step through here N times.
    1. Enter how many time allowed to execute through this action.
